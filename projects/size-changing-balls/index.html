[Jump to demo!]

This one was also me exactly copying line-to-line the fourth tutorial by Chris Courses on YouTube, "Interacting the Canvas | HTML5 Canvas Tutorial for Beginners - Ep. 4"... with almost no additions from my end.

It's a demonstration on how to use "event listeners". Event listeners is JavaScript's way of connecting user interations with certain functions defined by the developer. In this demo I am using the event listeners to find the position of the mouse and to know whether the user has resized the browser window.

The balls that fall inside the circle of 50px radius around the mouse pointer grows in size, and go back to their original size upon exit. This effect can the toggled by pressing any key.

Try playing around with the browser window size, you could in a very very stupid sense mimic a piston engine.
